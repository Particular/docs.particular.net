## Installing the ServiceControl Management Utility
#if-version [,5)
The ServiceControl Management Utility installer executable can be downloaded from [the Particular.net website](https://particular.net/downloads). This utility is launched as the final step in the installation process and is also available via the Windows start menu.
#end-if
#if-version [5,)
The ServiceControl Management Utility is a portable executable that can be downloaded from [the Particular.net website](https://particular.net/downloads) and can be installed and run from the location of choice.
#end-if

> [!NOTE]
> After installation, there is no ServiceControl instance running yet. Instances can be installed, upgraded, and removed using the ServiceControl Management Utility.
