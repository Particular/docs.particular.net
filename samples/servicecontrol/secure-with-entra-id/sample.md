This sample shows how to use YARP to define a reverse proxy to secure access to [ServicePulse](/servicepulse), and the resto of the [platform](/platform) tools, using Microsoft Entra ID.

This sample requires Docker and Docker Compose, or similar compatible technology, to run the platform tools, the YAPR reverse proxy, and two demo NServiceBus endpoints.
